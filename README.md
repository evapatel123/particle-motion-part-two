# 🌌 Particle Motion Part Two

An interactive particle motion simulation built with **HTML, CSS, and JavaScript**. This project demonstrates smooth animations, particle movement, and browser-based graphics using the HTML5 Canvas API.

---

## Live Demo

🔗 https://evapatel123.github.io/particle-motion-part-two/

---

## Features

- Smooth particle animations
- Real-time rendering
- Interactive visual effects
- Lightweight and responsive
- Built using vanilla JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API

---

## About the Project

This project explores browser-based graphics by simulating moving particles with JavaScript and the HTML5 Canvas API. It focuses on animation loops, object-oriented programming, and rendering techniques to create an engaging visual experience.
